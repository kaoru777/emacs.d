Set.prototype.values()

The values() method returns a new set iterator object that contains the values for each element in the Set object in insertion order.

Try it

Syntax

values()

Return value

A new iterable iterator object.

Examples

Using values()

const mySet = new Set();
mySet.add("foo");
mySet.add("bar");
mySet.add("baz");

const setIter = mySet.values();

console.log(setIter.next().value); // "foo"
console.log(setIter.next().value); // "bar"
console.log(setIter.next().value); // "baz"
