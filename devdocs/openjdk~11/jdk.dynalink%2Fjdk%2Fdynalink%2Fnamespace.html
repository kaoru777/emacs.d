Interface Namespace

All Known Implementing Classes:
StandardNamespace
public interface Namespace

An object that describes a namespace that is the target of a dynamic operation on an object. Every object can have one or more namespaces. Dynalink defines a set of standard namespaces with the StandardNamespace enum. Operations that need to specify a namespace they operate on can be expressed using NamespaceOperation.
