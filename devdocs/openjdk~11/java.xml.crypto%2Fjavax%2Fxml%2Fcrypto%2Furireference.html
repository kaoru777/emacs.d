Interface URIReference

All Known Subinterfaces:
DOMURIReference, Reference, RetrievalMethod 
public interface URIReference

Identifies a data object via a URI-Reference, as specified by RFC 2396.

Note that some subclasses may not have a type attribute and for objects of those types, the getType() method always returns null.

Since:
1.6
