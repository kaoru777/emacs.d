Module jdk.hotspot.agent

Defines the implementation of the HotSpot Serviceability Agent.

This module includes the jhsdb tool to attach to a running Java Virtual Machine (JVM) or launch a postmortem debugger to analyze the content of a core-dump from a crashed JVM.
