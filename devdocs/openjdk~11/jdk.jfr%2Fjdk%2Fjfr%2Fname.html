Annotation Type Name

@Target({TYPE,FIELD,METHOD})
@Retention(RUNTIME)
public @interface Name

Annotation that overrides the default name for an element (for example, when the default package for an event is not appropriate).

The name must be a valid identifiers in the Java language (for example, "com.example.MyEvent" for an event class or "message" for an event field).
