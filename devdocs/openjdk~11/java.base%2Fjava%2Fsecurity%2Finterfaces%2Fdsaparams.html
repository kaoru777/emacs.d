Interface DSAParams

All Known Implementing Classes:
DSAParameterSpec
public interface DSAParams

Interface to a DSA-specific set of key parameters, which defines a DSA key family. DSA (Digital Signature Algorithm) is defined in NIST's FIPS-186.
