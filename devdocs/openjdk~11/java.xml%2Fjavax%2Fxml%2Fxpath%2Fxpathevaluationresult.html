Interface XPathEvaluationResult<T>

Type Parameters:
T - the object type returned by the XPath evaluation.
public interface XPathEvaluationResult<T>

The XPathEvaluationResult interface represents the result of the evaluation of an XPath expression within the context of a particular node. The evaluation of an XPath expression can result in various result types as defined in XML Path Language (XPath) Version 1.0.
