Interface NewArrayTree

All Superinterfaces:
ExpressionTree, Tree 
public interface NewArrayTree
extends ExpressionTree

A tree node for an expression to create a new instance of an array. For example: 

new type dimensions initializers

   new type dimensions [ ] initializers

1.6
