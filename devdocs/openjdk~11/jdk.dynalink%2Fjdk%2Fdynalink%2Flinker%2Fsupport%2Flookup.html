Class Lookup

  • java.lang.Object
    • jdk.dynalink.linker.support.Lookup
public final class Lookup
extends Object

A wrapper around MethodHandles.Lookup that masks checked exceptions. It is useful in those cases when you're looking up methods within your own codebase (therefore it is an error if they are not present).

                                   Class<?> rtype,

Finds using findSpecial(Class, String, MethodType) a method on that lookup's class. Useful in classes' code for convenient linking to their own privates. It's also more convenient than findSpecial in that you can just list the parameter types, and don't have to specify lookup class.
