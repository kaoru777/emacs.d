Notifies this UI delegate to repaint the specified component. This method paints the component background, then calls the BasicListUI.paint(java.awt.Graphics, javax.swing.JComponent) method.

In general, this method does not need to be overridden by subclasses. All Look and Feel rendering code should reside in the paint method.
