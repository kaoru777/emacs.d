Enum Constant Details

OPEN

public static final ModuleDescriptor.Modifier OPEN
An open module. An open module does not declare any open packages but the resulting module is treated as if all packages are open.

AUTOMATIC

public static final ModuleDescriptor.Modifier AUTOMATIC
An automatic module. An automatic module is treated as if it exports and opens all packages.
API Note:
This modifier does not correspond to a module flag in the binary form of a module declaration (module-info.class).
